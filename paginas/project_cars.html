PROJECT CARS 2

Con Project Cars 2 la saga de conducción Project Cars suma una segunda entrega con 200 pistas diferentes repartidas en 50 localizaciones (tierra, grava, nieve, barro...), con ciclo día y noche y clima variable dependiendo de las estaciones que afecta a la conducción (Live Track 3.0), y que es uno de los grandes titulares de su mecánica de conducción. Ahora, por ejemplo, escoger los neumáticos es una decisión que hay que meditar con mucho cuidado y saber adaptarse después a los posibles cambios El simulador de velocidad busca marcar un antes y un después en este tipo de propuestas y, entre otras claves, para lograrlo el juego de Slightly Mad Studios incluye más de 200 vehículos de 40 clases diferentes y ocho disciplinas, un nuevo sistema de físicas y muchas mejoras para el online y eSports. Project Cars 2 (PC2) cuenta también con soporte para PS4 Pro, Project Scorpio y la Realidad Virtual. Por supuesto, eso sí, la edición para PC es la más mimada de cuantas presenta el estudio, con las mayores opciones de escalado visual y el mejor soporte para realidad virtual en caso de que nos interese esa opción para disfrutar de este SimRacing
